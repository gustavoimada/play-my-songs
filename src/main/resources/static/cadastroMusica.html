Spring PlayMySongs

Envie uma nova música para o sistema.

Preencha os dados da faixa, selecione o arquivo mp3 e envie tudo em um fluxo simples e visualmente consistente com a home.

Orientações

Antes de enviar

  • Use arquivos no formato `.mp3`
  • Preencha nome da música e do artista corretamente
  • Escolha um estilo para facilitar organização futura
  • Depois do envio, o backend retorna os dados cadastrados
Exemplo

lovemylife_pop_robbiewilliams.mp3

Você pode usar esse padrão visualmente como referência para o nome final do arquivo salvo.
